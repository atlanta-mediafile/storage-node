## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/atlanta-mediafile/storage-node.git
   ```
2. Run test
   ```sh
   go test -v ./...
   ```
3. Run node
   ```js
   go run .
   ```
